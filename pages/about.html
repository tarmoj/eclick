About

eClick

eClick is a wire-free visual click-track (or metronome) system, useful for ensembles, bands or other formations. No need for cables, headphones, extra amplifiers or mixers - just use wifi network and the eClick software.

There are two parts in the software - server (the main program that plays the click-track) and client (app for players that show information about bar numbers, beats, tempo etc).

The "click-track" (time, tempo, tempo changes, stops, fermatas etc etc) is written into a score in Csound format and played by the server. If necessary, also sound-clips or backtrack(s) can be started by the system in given moments. All necessary information is sent to clients (players) via wifi. Also extremely complex pieces can be encoded and played using the system. Every piece requires its own written out score. If the e-click score is done well, the playback can be started from any bar in the piece.

How it works

The software is written using QT development framework and Csound and is portable to almost any wide-spread platform (Linux, OSX, Windows, Android, iOS).

The software is open-source, available under the GNU GPL License. The source can be found here: LINK

eClick is currently under development and is not yet recommended for mission-critical performances.
